# runvard

runvard is a web-based control panel for managing Debian and Ubuntu servers from one private interface.
It provides live system monitoring, a browser terminal, file management, storage tools, network controls, security administration, and system maintenance features.
runvard includes Docker management, virtual machine controls, backup jobs, service management, logs, alerts, and server health dashboards.
It is designed for operators who want direct control over their own infrastructure without switching between many command-line tools.
runvard should be deployed on a trusted local network or behind a secure reverse proxy because it manages privileged host operations.
